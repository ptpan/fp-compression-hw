This is the project of floating point number compression hardware module. 
Two directories are included in this project:
	small_data_compression: standalone compression module
	xt_conn_trd: VC709 connectivity reference design with compression module integrated. There is also a dummy decompression module in this directory. 

How to open these files:
	You can open the .xpr file with Vivado to view both projects. We worked with Vivado 14.03 and both projects implements without timing violation.
